# Pre requisites for starting application

- Download docker
- Run docker locally
- Pull application from git, run it via docker compose



# Pre Requisites for JS tests
- npm@9.6.4

## Dependencies
- Supertest
- mocha
- chai
- uuid

**Install**
npm install supertest mocha chai uuid --save-dev


## Run tests 
Run following command on root of folder 

```
npm test 
```
